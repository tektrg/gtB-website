## Practical workflow: from summary → notes → action items

A summary that doesn’t change what you do next is just entertainment. Turn it into an execution plan:

```text
Convert this into an execution plan:
- 3 decisions to make
- 5 action items (with time estimates)
- 3 follow-up questions
Include timestamps for each supporting point.
```

## Common mistakes (over-trusting summaries, missing context, quoting issues)

Common mistakes:

- No timestamps → you can’t verify quickly. Ask for timestamps explicitly.
- Wall of text → ask for a table.
- Blind trust → click timestamps and verify the 2–4 key sections.
